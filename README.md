from command line, compile with 
$g++ main.cpp LaserScannerDriver.cpp -o filename.out
and launch with
$./filename.out

if you want to use debugger compile with
$ g++ -g main.cpp LaserScannerDriver.cpp -o filename.out
and launch with
$dbg filename.out
$run

to test memory leak
$valgrind ./filenameout
